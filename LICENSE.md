## 🛠️ Common Issues & Fixes

### ❓ "App not installed" error on Android
**Fix:** Make sure the original Roblox app is fully uninstalled before installing the
Arceus X Neo APK. Go to **Settings → Apps → Roblox → Uninstall**, reboot your device,
then retry the installation.

### ❓ Key verification stuck or not completing
**Fix:** Ensure you have a stable internet connection and are using the built-in browser
within Arceus X Neo for key verification. Switching to a different network (mobile data
vs. Wi-Fi) often resolves this. If still stuck, check **#key-issues** in the Discord.

### ❓ "Untrusted Developer" on iOS after install
**Fix:** Go to **Settings → General → VPN & Device Management**, find the Arceus X Neo
certificate entry, tap **Trust**, confirm, then fully restart your device before
launching the app.

### ❓ Floating GUI icon not visible in-game on Android
**Fix:** Grant the **"Display over other apps"** permission explicitly: go to
**Settings → Apps → Arceus X Neo → Permissions** and enable the overlay permission.
On MIUI/HyperOS devices, also check **"Background pop-up windows"** in App settings.

### ❓ Scripts from the hub not executing / returning errors
**Fix:** First confirm you're running the **arceus x neo apk latest update** — script
hub scripts are written for the current Roblox version and may break on outdated builds.
If updated and still failing, the specific game may have received an anti-cheat patch;
check the Discord **#script-status** channel.

### ❓ Executor crashes immediately on iOS launch
**Fix:** Delete the app completely, re-download the IPA, re-sign it with a fresh
certificate, and reinstall. Certificate expiry (common with free Apple Developer accounts
every 7 days) is the most frequent cause of this issue on iOS.

### ❓ "Failed to attach" on Windows
**Fix:** Right-click the Velocity executable and select **Run as Administrator**. Also
add the Arceus X Neo installation folder to your antivirus exclusion list — Windows
Defender frequently flags executor injection libraries as false positives.
